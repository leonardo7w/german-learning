# Established: based in Zurich, works in IT as a software architect/engineer

The learner lives in **Zurich, Switzerland** and works in **IT as a software
architect/engineer**. This anchors the "living and working abroad" mission concretely.

## Implications
- **Topic vocabulary stream = software/IT first:** software architecture, code review,
  deployment, requirements, agile/meeting language (stand-ups, design discussions, trade-offs).
- **Scenarios should be Swiss + tech-flavoured:** office/tech meetings, plus Swiss daily life
  (Behörden, Wohnung, Krankenkasse, ÖV/SBB).
- **Swiss German vs Hochdeutsch:** target Standard German (Hochdeutsch) for production — it's the
  written/formal language in Zurich and the basis of B2/C1. Treat *understanding* spoken Swiss
  German (Züritüütsch) as a later listening-comprehension goal, not a production target.
- Zurich tech workplaces are often partly English-speaking, so German here serves integration and
  broader opportunity — keep motivation framed that way, not as "you can't work without it."
